Cuberite Plugins

Various plugins created for the Cuberite server software.

  • CommandSpy

    Plugin for Cuberite that allows players to view commands executed by other players.

  • ExecuteCommandBlock

    A plugin that allows players to execute command blocks in Cuberite.

  • Extras

    Plugin for Cuberite that adds extra functionality to the Kaboom.pw server.

  • iControlU

    Plugin for Cuberite that allows players to control other players.

  • MobSound

    Plugin for Cuberite that allows you to broadcast a mob sound at your current location.

  • Weapons

    Plugin that adds weapons to a Cuberite server.
